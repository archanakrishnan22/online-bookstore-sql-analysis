# SQL Project: Online Bookstore Data Analysis

## Overview
This project involves analyzing an online bookstore database using SQL to extract insights related to customer behavior, book trends, and order patterns.

## Objective
To leverage SQL queries to identify patterns, trends, and relationships across multiple tables such as books, customers, and orders.

## Tools & Technologies
- SQL (PostgreSQL)

## Key Analysis Performed
- Joined multiple tables to analyze relationships between customers, books, and orders
- Performed aggregations to identify trends in sales and customer activity
- Filtered and segmented data to uncover meaningful patterns

## Key Insights
- Analyzed customer purchasing patterns across different regions
- Identified popular book categories and trends
- Examined order distribution and buying behavior

## Skills Demonstrated
- SQL Querying (JOINs, GROUP BY, Subqueries)
- Data Analysis
- Analytical Thinking
- Business Insight Generation
